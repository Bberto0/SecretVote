# SecretVote 🗳️

**SecretVote** è un prototipo di software per il voto digitale incentrato sulla sicurezza informatica, sull'ottimizzazione delle risorse e sulla tutela dei diritti democratici. Il progetto è stato selezionato come **Capolavoro** per il Portfolio dello studente (Maturità 2026).

---

## 📝 Descrizione del Progetto / Project Description

### Italiano
Il progetto consiste nella creazione di un potenziale software di voto digitale, dove l'utente ha garanzia di votazione segreta. Il codice assicura completa sicurezza ed una gestione ottimizzata per gestire possibili tentativi di frode o doppio voto, verificando l'unicità dell'elettore.

Il progetto rappresenta una perfetta sintesi tra innovazione tecnologica e la tutela dei diritti democratici. Sviluppare questo software mi ha permesso di riflettere ulteriormente sull'importante valore della segretezza del voto, elemento cruciale per garantire la piena libertà di espressione di ogni cittadino. Allo stesso tempo, questo lavoro dimostra come la transizione verso future votazioni digitali sia una strada percorribile, capace di unire la comodità dell'accesso da remoto a standard di sicurezza informatica speculari a quelli di un seggio fisico.

### English
The project involves developing a potential digital voting software that guarantees a secret ballot for the user. The code ensures full security and optimized management to prevent potential fraud or double-voting attempts by verifying voter uniqueness.

## 🚀 Funzionalità Principali / Key Features

1. **Garante dell'Anonimato:** Separazione logica tra l'identità dell'elettore e la preferenza espressa per garantire il voto segreto.
2. **Prevenzione del Doppio Voto:** Sistema di tracciamento e verifica dell'unicità dell'elettore per impedire votazioni multiple.
3. **Sicurezza del Codice:** Gestione ottimizzata dei flussi di dati per prevenire tentativi di manomissione.
